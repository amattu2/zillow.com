# Introduction

This is a simple React.js and Tailwind implementation of the <https://zillow.com>
homepage.

# Usage

```bash
git clone github.com/amattu2/zillow.com
```

```bash
cd zillow.com
```

```bash
npm install
npm start
```

Open <http://localhost:3000> in your browser.

# Requirements & Dependencies

- React 18+
- TypeScript 4.9.5+
- Tailwind CSS
